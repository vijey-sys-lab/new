# Devverse No-Skip Video Player (Frontend Only)

This is a fully frontend-only website that allows students to watch class videos (Day 1–20) without skipping forward. They must enter their name + Gmail before starting. A completion token is generated at the end of each video.

### Features
- No skipping forward (pause allowed)
- 1 video per day (Day 1–20)
- Admin panel for uploading/replacing videos (stored locally in browser IndexedDB)
- Automatic token generation (downloadable + copy)
- Fullscreen support
- Devverse branding included (logo + favicon)
- Pure HTML + JS (no backend required)
